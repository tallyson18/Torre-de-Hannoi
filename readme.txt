Este programa foi desenvolvido por Tallyson Lucas.
Tem como objetivo desenvolver um programa que utlize TAD's, onde deve-se cadastrar os jogadores que serão organizados em filas
e esta funcione de modo circular.
Serão criadas 3 pilhas de elementos, onde haverá, uma com 7 discos e duas vazias, e os jogadores devem mover os discos de tal forma que ao fim
tenham trasportados todos os discos de uma pilha para outra, utilizando uma terceira como auxiliar.
Entretanto não será permitido de forma alguma ficar um disco maior sobre um menor.
Será vencedor o usuário que tiver o menor número de jogadas quando o transporte completo for concluído.
Programa apresentado em Estrutura de Dados I  - 2ª Período de Ciência da Computação- Doctum,2018.
